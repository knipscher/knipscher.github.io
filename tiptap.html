Tiptap the Tadpole

Play it here on itch.io!

Tiptap the Tadpole is a local multiplayer game I created for Ludum Dare 48: Deeper and Deeper. One player controls Tiptap herself with the keyboard, while the other player puts together monsters with the mouse to chase her. I created everything in the game by myself from scratch in 48 hours, including code, visuals, and audio. Tiptap placed in the top 6% for Innovation, the top 8% for Graphics, and the top 25% overall.

The project files and code are publicly available here on my Github, in case you'd like to know what my code looks like (when I am working very, very fast).
